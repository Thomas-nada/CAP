# CAPSLOCK — CAP Editors Meeting Records

This folder contains the official meeting records of the CAP Editors. Each file documents the agenda, attendees, decisions, and action items from a scheduled editors meeting.

## File Naming

Meeting records follow this convention:

```
YYYY-MM-DD-meeting-records.md
```

## Template

```markdown
# CAP Editors Meeting — YYYY-MM-DD

**Attendees:** [Names / GitHub handles]
**Chair:** [Name]
**Scribe:** [Name]

---

## Agenda

1. [Item 1]
2. [Item 2]

---

## Decisions

- [Decision 1]
- [Decision 2]

---

## Action Items

| Action | Owner | Due |
|--------|-------|-----|
| [Task] | [@handle] | YYYY-MM-DD |

---

## Notes

[Free-form notes]
```
